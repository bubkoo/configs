# jest-config

> Shared jest config for my projects.

## Install

```sh
npm install --save-dev @bubkoo/jest-config
```


## Usage

`jest.config.js`

```js
module.exports = require('@bubkoo/jest-config');
```

`jest.config.ts`
```ts
import config from '@bubkoo/jest-config';
export default config;
```
