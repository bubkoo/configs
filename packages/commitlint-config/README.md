# commitlint-config

> Shared commitlint config for my projects.

## Install

```sh
npm install --save-dev @bubkoo/commitlint-config
```

## Usage

Add the following section into the `package.json` file.

```json
{
  "commitlint": {
    "extends": [
      "@bubkoo/commitlint-config"
    ]
  },
}
```
